Исследование на нормальные формы:

Drivers:
inn (A) ИНН
pas_series (B) серия паспорта
pas_num (C) номер паспорта
last_name (D)
first_name (E)
second_name (F)
birthday (G)
gos_num (H) госномер автомобиля

{A} - PK, {B, C} - Alternate Key
Все значения атомарны, отношение, очевидно, находится в 1НФ.

{A} -> {B, C, D, E, F, G, H}
{B, C} -> {A, D, E, F, G, H}
Слева нельзя опустить ни один атрибут, без потери части смысла => отношение находится во 2НФ.
Т.к. взаимозависимостей неключевых атрибутов нет => отношение находится в 3НФ.
Т.к. ключи не перекрываются и один из ключей простой, то НФБК совпадает с 3НФ.

Orders:
order_date (A)
order_time (B)
start_address (C)
finish_address (D)
passengers (E) количество пассажиров
inn (F)

{A, B, C, D} - PK, {A, B, F} - AK
Все значения атомарны, отношение, очевидно, находится в 1НФ.

{A, B, C, D} -> {E, F}
{A, B, F} -> {C, D, E}
Слева нельзя опустить ни один атрибут, без потери части смысла => отношения находятся во 2НФ.
Т.к. взаимозависимостей неключевых атрибутов нет => отношение находится в 3НФ.
Все условия НФБК соблюдаются. Т.к. детерминанты всех ФЗ являются потенциальными ключами =>
Отношение находится в НФБК.
